What is this?

Eurorack planner is an open source web application that helps you plan your eurorack setup. We welcome any contributions to the codebase, source code is available at: http://github.com/udbhav/eurorack-planner. Addititional information may be found on the thread at Muff Wiggler's.

Bugs/Feature Requests? Open an issue on Gitub.

Questions/Comments? Please email support@eurorackplanner.com

This project is developed and maintained by Udbhav Gupta.
